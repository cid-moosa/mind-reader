# Binary Card Guessing Game

## Description

A web-based card guessing game built with HTML, CSS, and JavaScript. The game uses a binary algorithm where the user answers whether their number appears on each card, allowing the program to correctly guess the chosen number.

## How It Works

* The user thinks of a number within a given range.
* A series of cards are shown, each representing a binary value.
* The user selects whether their number appears on each card.
* The game adds the binary values based on the responses and determines the final number.

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone this repository.
2. Open the `index.html` file in any modern web browser.
