# Number Guessing Game 🔮

## Description
A magical web-based card guessing game built with vanilla HTML, CSS, and JavaScript. It uses a clever binary algorithm to read your mind! By asking you a few simple "yes or no" questions about whether your number appears on specific cards, the program uses binary place values to magically predict the exact number you were thinking of.

## Live Demo
✨ **[Play the Live Demo Here!](https://cid-moosa.github.io/number-guessing-game/)** ✨

## How It Works
1. **Pick a number:** You mentally choose any number within a given maximum range (e.g., 1 to 31).
2. **Answer the cards:** The game shows you a series of "cards", each containing a list of numbers. You simply answer whether your number is on the card.
3. **The Reveal:** Behind the scenes, each card represents a binary place value (1, 2, 4, 8, 16...). The game adds together the values of the cards where you answered "yes" and instantly predicts your number using binary logic!

## Technologies Used
- **HTML5** for semantic structure
- **CSS3** for modern, responsive, and dynamic UI (including multiple themes and micro-animations)
- **JavaScript (Vanilla)** for the binary guessing logic and state management

## Running Locally
Since it is fully front-end, no server is required:
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
3. Enjoy the magic!
